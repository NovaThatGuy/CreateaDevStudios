Task Master

What is Task Master?

First of all Task Master is a placeholder name for the Android application we are working on. However Task Master is a fairly fitting name as the application's purpose is to track and log a user's tasks. They can sort tasks by importance and set when tasks will move to being more important if they are left undone for a set period of time. The progress of Task Master will be publically available through Github soon, if you are interested in seeing it's progress feel free to check it out there.
